# CWE Top-25
> Community-Developed List which ranks the Top-25 most common weaknesses found in [[Website Applications\|Websites]] created by a non-profit organisation by the name of [[MITRE\|MITRE]].

CWE stands for Common Weakness Enumeration.
Helps developers focus on Areas that lead to exploitability issues.

The list for Vulnerabilities is [[CVE Top-25\|CVE Top-25]].
