# Blockchain
> Name of a technology where data is written on blocks which are a part of a [[Decentralised Network\|Decentralised]] Network of computers. Each block also contains the address of the previous block which forms a chain of blocks - this chain is also called as Blockchain & this is how the name for the overaching technology was also derived.

Blockchain is a technology which may be used in other things but has been brought to light and made famous particularly in the form of [[Cryptocurrency\|Cryptocurrency]] like [[Bitcoin (BTC)\|Bitcoin (BTC)]]. Many still know Blockchain only by its connections with Crypto.

Not every [[Decentralised Network\|Decentralised Technology]] is Blockchain by the way... some are distinct - such as the [[NOSTR\|NOSTR]] protocol.
