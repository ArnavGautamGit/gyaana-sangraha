# Zero-Attack Proxy (ZAP)
> A [[Vulnerability Scanning\|Vulnerability Scanning]] tool created by [[OWASP\|OWASP]] to be used during [[Dynamic Application Security Testing (DAST)\|Dynamic Application Security Testing (DAST)]]. It can only be used as normal users not as root user.

To download it, visit its website.
On most (if not all CLI, it is called `ZAProxy` or `zaproxy`).
