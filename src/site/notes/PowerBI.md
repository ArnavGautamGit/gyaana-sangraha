# PowerBI
> [[Data Analysis\|Data Analysis]] tool used for Data Visualisation made by Microsoft. It natively supports Microsoft Excel spreadsheets and [[SQL\|SQL]] Databases as sources to import the data and begin the visualisation process. 

Although it is from 2022, I learnt the basics of PowerBI from [This YouTube tutorial](https://youtu.be/9tF1IrfLflg?si=GdYC2O-Pdu4kBGWX).
