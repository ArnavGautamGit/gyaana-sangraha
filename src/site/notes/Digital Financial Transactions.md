# Digital Financial Transactions
> Transactions (Exchange) of [[Fiat Money\|Money]] via digital means.

Financial Transactions can occur Digitally in two ways: 
1. using the same currency via digital means like [[National Electronic Financial Transactions (NEFT)\|NEFT]], [[Real Time Gross Settlement (RTGS)\|RTGS]], [[Instant Money Payment System (IMPS)\|IMPS]], [[Unified Payment Interface (UPI)\|UPI]] or [[Plastic Money\|Plastic Money]]
2. using creating a digital version of curencies such as [[Central Bank Digital Currency (CBDC)\|Central Bank Digital Currency (CBDC)]]
